🔧 Режим обслуживания

Просмотреть phpInfo()

Внимание!, Вашему вниманию представлена диагностиечская страница, которая по умолчанию расположена в контейнере.

Возможны следующие варианты запуска контейнера:

Вариант 1. Контейнер будет доступен только на хосте, на котором он запущен, по адресу http://localhost:8080: 

docker run -d -p 8080:80 alt-p10-web:1.0.0

Вариант 2. Контейнер будет доступен по адресу "http://192.168.0.111:8080" для других компьютеров локально-вычислительнос сети, где "192.168.0.101" IP-адрес компьютера в сети, на котором запущен контейнер: 

docker run -d -p 0.0.0.0:8080:80 alt-p10-web:1.0.0

Вариант 3. Контейнер будет доступен по адресу "http://192.168.0.111:8080" для других компьютеров локально-вычислительнос сети, где "192.168.0.101" IP-адрес компьютера в сети, на котором запущен контейнер, с прикреплением к контейнеру каталога php-скриптов: 

docker run -d -p 0.0.0:8080:80 -v ./attach_web:/var/www/html alt-p10-web:1.0.0

При запуске контейнера необходимо навигационно необходимо находиться в одном каталоге с каталогом "./attach_web" или вместо него указать полный путь до другого каталога.
